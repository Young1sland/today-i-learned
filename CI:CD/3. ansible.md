### Infrastructure as Code(IaC)
- 시스템,하드웨어 또는 인터페이스의 구성 정보를 파일(스크립트)을 통해 관리 및 프로비저닝
- IT 인프라스트럭처, 베어 메탈 서버 등의 물리 장비 및 가상 머신과 관련된 구성 리소스를 관리
- 버전 관리를 통한 리소스 관리
    > 베어 매탈 : 어떠한 운영 체제나 소프트웨어가 설치되지 않은 순수한 하드웨어 상태를 의미   
    > 베어 메탈 서버 : 베어 메탈 서버는 가상화 레이어나 하이퍼바이저 없이 직접 물리적 서버 하드웨어에 운영 체제가 설치되는 서버를 의미

### Ansible
- 여러개의 서버를 효율적으로 관리할 수 있게 해주는 환경 구성 자동화 도구
- client node들은 별도의 Agent 설치 없이 python 모듈,ssh 가지면 연결 가능함.
- 환경 설정 파일 : /etc/ansible/ansible.cfg
- Ansible에서 접속하는 호스트 목록 : /etc/ansible/hosts
