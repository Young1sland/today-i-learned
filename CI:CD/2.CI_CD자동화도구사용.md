### 플러그인 설치 확인
1. Jenkins 관리 -> Plugin Manager
   - github Integration,maven Integration 확인
2. Jenkins 관리 -> Global Configuration    
   - docker 접속하여 git, mvn 명령어 사용가능한지 확인
   - 안될 경우 설치 필요. Global Configuration에서 git,Mvn 탭에서 설치 설정

### 실습 Git에서 코드 가져와서 빌드 하기
1. 새로운 아이템 -> Maven Project (없다면 Maven 설치 확인할 것)
2. 소스 코드 관리 탭 -> git 선택 -> 대상 repository URL 입력    
   https://github.com/joneconsulting/cicd-web-project
3. 소스 코드 관리 탭 -> Branches to build -> branch 이름이 일치하는지 확인
4. Build 탭 -> Root POM pom.xml -> Goals and options : clean compiler:compile package    
   - 빌드 이력 있으면 지움. 빌드. 패키지 파일 생성(패키지 파일 형식은 pom.xml의 packaging으로 결정)
5. 지금 빌드 -> 로그 확인
6. 작업 공간 탭으로 가서 파일 확인, docker 접속하여 target 폴더의 war 확인
```shell
$ docker exec -it jenkins-server bash
```
