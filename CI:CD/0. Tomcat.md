### 톰캣 설치
- https://tomcat.apache.org/ 
- tomcat core 9.x 다운로드

### 서버 연결 시작/종료
```sh
# 서버 시작
$ ./bin/startup.sh

# 서버 종료
$ ./bin/shutdown.sh
```

### 서버 연결 정보
./conf/server.xml 
```xml
<Connector port="8080" protocol="HTTP/1.1" address="192.168.1.100" 
               connectionTimeout="20000"
               redirectPort="8443" />
```

### 로그 확인
logs/catalina.out
```sh
 # tail -f 새로운 내용이 추가될 때마다 그 내용을 터미널에 출력
$ tail -f logs/catalina.out
```

### 사용자 계정 설정
conf/tomcat-users.xml   
```xml
<!-- conf/tomcat-users.xml -->
<role rolename="manager-gui"/>
<role rolename="manager-script"/>
<role rolename="manager-jmx"/>
<role rolename="manager-status"/>
<user username="admin" password="admin" roles="manager-gui, manager-script, manager-jmx, manager-status"/>
<user username="deployer" password="deployer" roles="manager-script"/>
<user username="tomcat" password="tomcat" roles="manager-gui"/>
```

### 톰캣 웹앱 관련 설정
webapps/manager/META-INF/context.xml   
webapps/host-manager/META-INF/context.xml
```xml
<!-- 톰캣 매니저에 접근할 수 있는 IP 제한, 주석 처리하거나 접근 원하는 IP 추가 -->
<Valve className="org.apache.catalina.valves.RemoteAddrValve"
        allow="127\.\d+\.\d+\.\d+|::1|0:0:0:0:0:0:0:1|192\.168\.1\.100" />
```
