### 실습 위해 docker 통한 ssh server 실행하기
1. hub.docker.com에서 edowon0623/docker-server 검색
2. 이미지 다운 받기
   ```shell
   $ docker pull edowon0623/docker-server:m1
   ```
3. ssh 서버 docker 실행
   ```shell
    # -- previleged : root 권한 획득
    $ docker run --privileged --name docker-server -itd -p 10022:22 -p 8082:8080 -e container=docker -v /sys/fs/cgroup:/sys/fs/cgroup:rw --cgroupns=host edowon0623/docker-server:m1 /usr/sbin/init

    # docker 중지 및 삭제
    $ docker stop docker-server
    $ docker rm docker-server
   ```
4. ssh 서버에 접근   
   ```sh
   $ ssh root@localhost -p 10022 # (password : P@ssw0rd)
   ```
   접근 시 Host Key verification failed가 뜨는 경우    
   ssh 접속 정보는 {HOME}/.ssh/known_hosts에 기록 됨. 여기서 해당 기록을 삭제하자.
5. 서버 내부에서 docker 시작
   ```shell
   $ systemctl enable docker
   $ systemctl start docker
   $ systemctl status docker

   # 만약 start 시 에러가 나면 selinux를 disable
   # /etc/sysconfig/docker의 OPTIONS='--selinux-enabled=false'

   # 필요한 패키지 설치
   $ yum install -y iptables net-tools

   # 참고
   # sed 명령어 : 파일의 특정 내용을 치환하는 용도로 사용. 
   # ex) 's/A/B/g' A를 B로 치환.

   # ex) /etc/sysconfig/docker-storage 파일에서 overlay2를 vfs로 치환
   $ sed -i -e 's/overlay2/vfs/g' /etc/sysconfig/docker-storage
   ```
