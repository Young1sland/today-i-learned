# 액세스 토큰 관리

## 로컬 스토리지

### 로컬 스토리지
- 웹 스토리지를 이용해 브라우저에 key-value 형태로 데이터 저장 가능
- 웹 스토리지에는 세션 스토리지, 로컬 스토리지가 있음. 세션 스토리지는 브라우저 닫으면 사라지지만 로컬 스토리지는 유지 됨
- 사용자가 브라우저를 재시작할 때마다 로그인하게 하고 싶으면 세션 스토리지, 재시작해도 로그인 상태 유지하고 싶으면 로컬 스토리지를 사용하면 됨
- 로컬 스토리지는 도메인마다 따로 저장 됨. JS는 다른 도메인의 로컬 스토리지 알지 못함.
```js
localStorage.setItem('hello', 'world')
localStorage.getItem('hello')
```
