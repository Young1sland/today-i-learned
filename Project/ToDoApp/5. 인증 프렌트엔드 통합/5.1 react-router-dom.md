# react-router-dom

## react-router-dom 설치
```$ npm install react-router-dom@6.30.0```

## react-router-dom의 역할
- 클라이언트 사이드 라우팅은 서버로 어떤 요청을 날리지 않고 모든 라우팅은 클라이언트 코드가 해결
- 리액트 앱은 브라우저에서 필요한 모든 리소스를 갖고 있고 주소창에 url 입력 하면 리액트 라우터가 이를 가로채 해석한 뒤 컴포넌트를 렌더링한다.
- 클라이언트 사이드 라우팅은 보편적인 서버사이드 라우팅과 달라 라이브러리가 필요함.

### ```<BrowserRouter>```
브라우저가 관리하는 히스토리를 사용해 브라우저와 리액트 사이의 URL을 동기화하여 뒤로가기 버튼도 지원 됨.

### ```<Routes>와 <Route>```
- ```<Route>```는 실제 경로를 지정해 주기 위한 컴포넌트
- ```<Routes>``` : 여러개의 Route를 관리하고 적합한 것을 찾아주는 컴포넌트
```js
//AppRouter.js
function AppRouter() {
    return (
        <div>
            <BrowserRouter>
                <Routes>
                    <Route path="/" element={<App />} />
                    <Route path="login" element={<Login />} />
                </Routes>
            </BrowserRouter>
        </div>
    );
}
export default AppRouter;
```
