### 인증서 요청
- AWS console -> Certificate Manager -> 인증서 요청
- 도메인 이름 추가에 *.<도메인이름> 이렇게하면 서브도메인도 하나의 인증서 같이 사용 가능

### SSL/TLS 인증서 발급 받기
- aws console -> Certificate Manger -> 인증서 요청
- *.<도메인이름>으로 하면 app.<도메인이름>, api.<도메인이름>에 하나의 인증서 사용 가능
- Route53에서 도메인 구매하면 DNS 검증으로 쉽게 도메인 확인 작업 가능
- DNS검증은 CNAME으로 하므로 Route53에서 레코드 생성하기까지 하고 좀 기다리면 발급 됨
### App에 HTTPS 설정
- aws console -> Elastic Beanstalk -> App 선택-> 구성 -> 로드밸런서
- 로드밸런서 편집 -> 리스너 추가 -> SSL 인증서 선택 -> 적용 
  *SSL 정책은 선택 안해도 되는 듯
