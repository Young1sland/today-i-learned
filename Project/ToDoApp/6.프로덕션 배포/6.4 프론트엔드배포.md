### Swap 파일 생성
- Swap이란 메모리가 부족할 때, 사용하지 않는 메모리의 일부를 파일로 옮겨 메모리 확보하는 기술
- t2/t3 계열의 메모리가 작은 인스턴스 사용 시 Out Of Memory 방지 위해 Swap 파일 생성
```shell
# .platform/hooks/prebuild
#!/bin/bash
SWAPFILE=/var/swapfile

if [ -f $SWAPFILE ]; then
    echo "$SWAPFILE found, skip"
    exit;
fi

/bin/dd if=/dev/zero of=$SWAPFILE bs=1M count=1024
/bin/chmod 666 $SWAPFILE
/sbin/mkswap $SWAPFILE
/sbin/swapon $SWAPFILE
```

### package.json에 engine version 추가
일래스틱 빈스톡도 같은 nodejs 버젼 사용 위해 엔진 버젼 추가
```json
{"engines":{"node":"16.14.0"},}
 ```

### NodeJS 사용하는 일래스틱 빈스톡의 경우 커밋되지 않은 변경 사항들은 배포되지 않으므로 커밋 필요

### 배포
```shell
$ eb create --elb-type application --instance-type t3.micro #micro하니까 OOM Error 발생하네
$ eb deploy #수정 후 재배포할 때
```
