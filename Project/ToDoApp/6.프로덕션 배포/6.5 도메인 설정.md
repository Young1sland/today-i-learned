### Route53
- Route53 도메인 생성-> 호스팅 영역 생성 -> 서브 도메인 생성
- 레코드
  - SOA(Start Of Authority) 레코드 : 관리자의 정보를 가짐
  - NS(Name Server) 레코드 : 해당 도메인의 IP를 물어볼 서버를 가짐
  - A 레코드 : 해당 도메인을 특정 IP 또는 다른 도메인으로 연결하는 레코드
- 로드 밸런서에 app.XX, api.xx 서브 도메인 생성해서 연결    
  트래픽 라우팅 대상 별칭 선택 -> Application/Classic Load Balancer    
  EC2-> 로드밸런서 -> 태그에서 연결된 애플리케이션 확인 가능
