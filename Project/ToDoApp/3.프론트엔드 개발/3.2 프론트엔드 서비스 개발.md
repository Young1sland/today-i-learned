### 리액트 훅
리액트가 제공하는 기능과 상태변수를 사용할 수 있도록 제공하는 일련의 함수

### useState
- 함수형 컴포넌트에서 상태 변수 사용할 수 있도록 해줌
``` const [추적할 Object, 상태 변경 함수] = useState(초기상태값);```

### JS로 된 변수를 JSX, HTML에서 사용할라면 {}로 묶어준다.

### 상위 컴포넌트에서 하위컴포넌트로 함수를 props로 넘기기
```js
//App.js
function App() {
  const [items, setItems] = useState([]);
  
  const addItem = (item) => {
    item.id = "ID-" + items.length;
    item.done = false;
    /*리액트는 레퍼런스를 기준으로 재렌더링한다. 배열의 레퍼런스는 배열에 값을
    추가하더라도 변하지 않는다. 그래서 배열에 item을 추가할 때마다 새배열을 생성해준다.*/
    setItems([...items, item]);
    console.log("items : ", items);
  }
  ..
  return (
    <div className="App">
      <Container maxWidth="md">
    {/*   하위 컴포넌트로 함수를 props로 넘긴다.*/}
        <AddTodo addItem={addItem} />
       ..
      </Container>
      {todoItems}
    </div>
  );
}

//AddTodo.js
```
