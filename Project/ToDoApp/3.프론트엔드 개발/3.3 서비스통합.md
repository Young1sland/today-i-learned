### CORS 설정
```java
@Configuration
public class WebMvcConfig implements WebMvcConfigurer {
    private final long MAX_AGE_SECS = 3600;

    @Override
    public void addCorsMappings(CorsRegistry registry){
        registry.addMapping("/**") //모든 경로에 대해
                .allowedOrigins("http://localhost:3000") //설정된 Origin에 대해
                .allowedMethods("GET","POST","PUT","PATCH","DELETE","OPTIONS") //허용 메소드
                .allowedHeaders("*")
                .allowCredentials(true)//모든 헤더와 인증에 관한 정보 허용
                .maxAge(MAX_AGE_SECS);
    }
}
```

### Effect Hook 이용한 Todo 리스트 초기화
- 리액트는 HTML DOM 트리의 다른 버젼인 ReactDOM(Virtual DOM)을 가지고 있고 컴포넌트의 상태가 변하면 컴포넌트 함수를 다시 호출하여 변경된 부분을 다시 렌더링한다.
- 백엔드 API 호출하여 Todo 아이템 리스트를 그냥 초기화할 경우 상태가 바뀜에 따라 리액트는 App()을 호출, 계속해서 재렌더링을 시도하여 무한루프에 빠진다.
- useEffect(콜백 함수, 디펜던시 배열) 사용    
  첫 렌더링(또는 마운팅)이 일어났을 때, 이후에는 오브젝트 값이 변할 때마다 콜백 함수 호출한다.    
  ```js
  function App() {
    const [items, setItems] = useState([]);

    useEffect(() => {
        const requestOptions = {
        method: "GET",
        headers: { "Content-Type": "application/json" }
        }
        //useEffect에 빈 배열을 넘겨 한번만 실행되게 함.
    
        fetch("http://localhost:8080/todo", requestOptions)
        .then((response) => response.json())
        .then((response) => {
            setItems(response.data);
        }, (error) => { });  
    }, []);
  }
  ```

### Javascript Promise
함수를 실행 후 Promise 오브젝트에 명시된 사항들을 실행시키겠다는 약속
- Pending : 오퍼레이션이 끝나길 기다리는 상태
- resolve : 오퍼레이션이 성공적으로 끝난 상태로 then의 함수 실행
- reject : 오퍼레이션 중 문제가 생기는 경우 then의 두번째 인자로 들어오는 reject 함수 콜함
- 실행 중 예외 발생하면 catch 부분이 실행 됨
```js
//XMLHttpRequest Promise 구현
```
