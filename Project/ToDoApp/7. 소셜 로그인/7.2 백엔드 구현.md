### 소셜 로그인
- 싱글 사인 온(Single Sign On) : 하나의 아이디로 여러개의 독립된 App에 로그인할 수 있는 인증 매커니즘
- 소셜 로그인은 싱글 사인 온의 일종이라 할 수 있고 구현 방법 중 하나가 OAuth2임

### 소셜 로그인(Github) 사이트에서 클라이언트 App 생성하기
- 리소스 서버 및 인가 서버는 클라이언트 App을 알고 있어야 함.
- github-> Settings -> Developer settings -> OAuth Apps->Register
- Homepage URL : http://localhost:8080 (Backend URL)
- Authorization callback URL : http://localhost:8080/oauth2/callback (Backend URL의 Callback endpoint)
- Client ID, Secret Key 기록
