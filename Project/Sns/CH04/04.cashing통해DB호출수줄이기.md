### Redis Configuration class 생성
```java
@Configuration
@EnableRedisRepositories
@RequiredArgsConstructor
public class RedisConfiguration {
    /** RedisProperties
     *  @ConfigurationProperties(prefix = "spring.redis")
     *  public class RedisProperties {     *
     *
     *  Database index used by the connection factory.
     *	private int database = 0;
     *  application.yaml파일에 설정하고 싶은 properties를 spring.redis.XX 형태로 입력한다.
     */
    private final RedisProperties redisProperties;

    @Bean
    public RedisConnectionFactory redisConnectionFactory(){
        RedisURI redisURI = RedisURI.create(redisProperties.getUrl());
        //Redis 연결하는 라이브러리는 여러가지가 있으나 Lettuce를 주로 사용한다.
        org.springframework.data.redis.connection.RedisConfiguration configuration = LettuceConnectionFactory.createRedisConfiguration(redisURI);
        LettuceConnectionFactory factory = new LettuceConnectionFactory(configuration);
        factory.afterPropertiesSet(); //intialze함
        return factory;
    }

    @Bean
    public RedisTemplate<String, User> userRedisTemplate(RedisConnectionFactory redisConnectionFactory) {
        RedisTemplate<String, User> redisTemplate = new RedisTemplate<>();
        //redisTemplate.opsForValue() get,set할 수 있게 해주는 메소드
        redisTemplate.setConnectionFactory(redisConnectionFactory);
        redisTemplate.setKeySerializer(new StringRedisSerializer()); //key로 String type 사용했으므로
        redisTemplate.setValueSerializer(new Jackson2JsonRedisSerializer<User>(User.class));
        //value는 object이므로
        return redisTemplate;
    }

}
```

### UserCacheRepository
```java
@Slf4j
@Repository
@RequiredArgsConstructor
public class UserCacheRepository {
    private final RedisTemplate<String, User> userRedisTemplate;

    //일정 시간 지나면 Expire 시키기 위함
    private final static Duration USER_CACHE_TTL = Duration.ofDays(3);

    public  void setUser(User user){
        String key = getKey(user.getUsername());
        log.info("Set User to Redis {},{}", key,user);
        userRedisTemplate.opsForValue().set(key,user, USER_CACHE_TTL);
    }

    public Optional<User> getUser(String userName){
        String key = getKey(userName);
        User user = userRedisTemplate.opsForValue().get(key);
        log.info("Get data from Redis {},{}", key, user);
        return Optional.ofNullable(user);
    }

    //userName에 prefix를 붙여 key를 가져오는 함수
    private String getKey(String userName)
    {
        //구별 위해 prefix를 붙여준다.
        return "USER: "+ userName;
    }
}
```

### JwtTokenFilter Error 수정
```java
public class AuthenticationConfig extends WebSecurityConfigurerAdapter {
    @Override
    public void configure(WebSecurity web) throws Exception {
        //정규 표현식으로 필터링. /api/로 시작하는 패스들만 인증거침(보안필터 거침). 아닌 것들은 무시

        /*.regexMatchers(..);
           (JwtTokenFilter : Error occurs while getting header.) 해결 위함
           HttpSecurity Configure에서 join,login permitAll 해줬으나 JwtTokenFilter는 거치므로 Error 발생
           join,login 시 보안필터 거치지 않도록 함
        */
        web.ignoring().regexMatchers("^(?!/api/).*")
                .regexMatchers("/api/*/users/join", "/api/*/users/login");
    }

    @Override
    protected void configure(HttpSecurity http) throws Exception {
        http.csrf().disable()
                .authorizeRequests()
                .antMatchers("/api/*/users/join", "/api/*/users/login").permitAll()
                .antMatchers("/api/**").authenticated()
```
