- spirng boot는 어플리케이션이 실행될 때 자동으로 application.properties, application.yml을 찾으며 properties가 더 나중에 로딩된다.
```yml
### ---로 한 파일에서 구분해서 정의할 수도 있다.
spring:
  profiles:
    active: local # default 사용 profile
    group:
      local:
        - common # local 사용 시 common도 사용
      prod:
        - common # prod 사용 시 common도 사용
---
spring:
  profiles:
    activate:
      on-profile: common
---
spring:
  profiles:
    activate:
      on-profile: local
---
spring:
  profiles:
    activate:
      on-profile: prod
---
# : The following 2 profiles are active: "local", "common"
```
