### IntelliJ 환경설정
- Preferences -> Build,Exception,Deployment-> Gradle을 IntelliJ IDEA 변경
- Preferences -> plugins -> Spock Framework Enhancedments,Handlebars/Mustache 설치
- Preferences -> Annotation Processors -> Enable annotation processsing
- springboot 버젼 변경할려면 build.gradle에서 변경
```gradle
plugins {
    id 'java'
    id 'org.springframework.boot' version '2.6.7'
    id 'io.spring.dependency-management' version '1.0.15.RELEASE'
}
```
