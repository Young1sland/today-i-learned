
### 컨테이너 생성 및 시작
1. docker Login
```shell
$ docker login
```
2. jar 생성
```gradle
# build.gradle
# jar 파일 이름 변경
bootJar {
    archiveFileName = 'app.jar'
}
```
```shell
$ ./gradlew build
# 기본 이미지 생성 경로는 build/libs/
# *plain이 붙어 있는 jar는 의존성을 포함하지 않는 파일
```
3. Dockerfile 작성
프로젝트의 루트 경로에 Dockerfile 생성
```Dockerfile
FROM openjdk:11
ARG JAR_FILE=build/libs/app.jar
COPY ${JAR_FILE} ./app.jar
ENV TZ=Asia/Seoul
ENTRYPOINT ["java","-jar","./app.jar"]
```

1. docker build 및 실행
```shell
$ docker build -t leeiubba/app-project-test .
$ docker iamges
$ docker run leeiubba/app-project-test -p 8080:8080 .
$ docker ps
$ docker exec -it [컨테이너 이름 또는 id] bash
$ docker stop [컨테이너 이름 또는 id]
$ docker inspect [컨테이너 이름 또는 id]
```
