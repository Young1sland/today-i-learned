### Official Package repository
https://pub.dev

### pubspec.yaml
- project 설정 파일로 assets,fonts,library dependencies를 설정할 수 있음
- VS code에서는 이 파일에 dependency를 추가하면 패키지 설치 함. 상단의 다운로드 아이콘을 눌러도 같은 동작 수행

### Http GET
```dart
Future<Response> get(Uri url, {Map<String, String>? headers})
```
- Future는 나중에 return 함을 나타냄.
- Future 리턴하는 함수 사용하는 함수는 비동기(async)로 정의하고 사용 시 await 키워드 붙여주자.
