### 프로젝트 생성
```shell
$ flutter create {project_name}
```
### 실행
- 진입점 lib/main.dart
- VS Code에서 실행 시 우측 상단 플레이 버튼 사용. 우측 하단에 실행할 대상 platform 선택
```shell
$ flutter run # command로 실행 시
```

### HelloWorld 샘플
- Flutter은 Widget을 합쳐서 만들어짐
```dart
void main() {
  runApp(App());
}

class App extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    //MaterialApp,CupertinoApp 둘 중 하나를 리턴
    //CupertinoApp : iOS 스타일
    //MaterialApp : google 스타일, google에서 만들었으므로 material스타일이 더 좋은 편..
    //class를 만들 때마다 끝에 ','를 붙여주자. VS Code의 formatting 위함
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Hello flutter'),
        ),
        body: Center(
          child: Text('Hello world!'),
        ),
      ),
    );
  }
}
```
