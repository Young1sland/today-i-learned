### Transform
사이즈를 변경하거나 위치를 변경
```dart
Transform.scale( //크기 변경
    scale: 2.2, //2.2배
    child: Transform.translate( //위치 조정
        offset: const Offset(-5, 12), //위치를 offset만큼 조정
        child: const Icon(
            Icons.euro_rounded,
            color: Colors.white,
            size: 88,
        ),
    ),
)

//컨테이너 영역을 초과하는 부분은 잘라내도록 설정
Container(
    clipBehavior: Clip.hardEdge,
    ...
);
```
