### MainAxis,CrossAxis
- Column의 MainAxis는 vertical, Row의 MainAxis는 horizontal

### SizedBox
고정 크기의 박스 생성, 공백 주기 위해 사용
```dart
    SizedBox(
        height: 80,
    ),
```

### Padding
- 패딩 설정
```dart
// EdgeInsets.all() : 4방향 모두 40 pixel 마진
// EdgeInsets.only(left: 40.0) : 왼쪽만 40 pixel 마진
// EdgeInsets.symmetric(horizontal: 40) : 좌우로 40 pixel 마진
Padding(
    padding: EdgeInsets.symmetric(horizontal: 40),
    child: ...
)
```

### Color 설정
```dart
Text('Welcome back',
    style: TextStyle(
        color: Colors.white.withOpacity(0.8),
        fontSize: 18,
    )),
//Color.fromRGBO(24,24,24,1) //red,green,blue, opacity
//Color(0xFF181818)
```

### 둥근 버튼 만들기 (Container)
- Container, decoration 사용
```dart
  decoration: BoxDecoration(
    color: Colors.amber,
    borderRadius: BorderRadius.circular(45),
  ),
  child: ...
 );
```

### const
- 컴파일 타임에 값을 알고 있고 변경되지 않는 값을 const로 지정.
- const로 지정해두면 런타임이 아닌 컴파일 타임에 값을 산정할 수 있음
- 단 const인지 아닌지 확인이 어려우므로 자동으로 처리하도록 VS Code에 Setting 추가해 두자.
```json
//settings.json (F1 -> open user setting)
 "editor.codeActionsOnSave": {
        "source.fixAll": true
  },
 "dart.previewFlutterUiGuides": true, 
```

### Flutter UI Guide 라인 설정
```json
//settings.json (F1 -> open user setting)
 "dart.previewFlutterUiGuides": true, 
```

### 예제
```dart

void main() {
  runApp(App());
}

class App extends StatelessWidget {
  const App({super.key});

  @override
  Widget build(BuildContext context) {
    //MaterialApp,CupertinoApp 둘 중 하나를 리턴
    //CupertinoApp : iOS 스타일
    //MaterialApp : google 스타일, google에서 만들었으므로 material스타일이 더 좋은 편..
    //class를 만들 때마다 끝에 ','를 붙여주자. VS Code의 formatting 위함
    return MaterialApp(
      home: Scaffold(
        backgroundColor: const Color(0xFF181818),
        body: Padding(
          padding: const EdgeInsets.symmetric(
            horizontal: 40,
          ),
          child: Column(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              //SizedBox : 고정 크기의 박스 생성. 위의 공백 위함
              const SizedBox(
                height: 80,
              ),
              Row(
                mainAxisAlignment: MainAxisAlignment.end, //수평 방향 정렬
                children: [
                  Column(
                    crossAxisAlignment: CrossAxisAlignment.end,
                    children: [
                      const Text('Hey, Selena',
                          style: TextStyle(
                            color: Colors.white,
                            fontSize: 28,
                            fontWeight: FontWeight.w800,
                          )),
                      Text('Welcome back',
                          style: TextStyle(
                            color: Colors.white.withOpacity(0.8),
                            fontSize: 18,
                          )),
                    ],
                  )
                ],
              ),
              const SizedBox(
                height: 120,
              ),
              Text(
                'Total balance',
                style: TextStyle(
                  fontSize: 22,
                  color: Colors.white.withOpacity(0.8),
                ),
              ),
              const SizedBox(
                height: 10,
              ),
              const Text(
                '\$5 194 482',
                style: TextStyle(
                  fontSize: 44,
                  fontWeight: FontWeight.w600,
                  color: Colors.white,
                ),
              ),
              const SizedBox(
                height: 30,
              ),
              Row(
                children: [
                  Container(
                    decoration: BoxDecoration(
                      color: Colors.amber,
                      borderRadius: BorderRadius.circular(45),
                    ),
                    child: const Padding(
                      padding: EdgeInsets.symmetric(
                        vertical: 20,
                        horizontal: 40,
                      ),
                      child: Text('Transfer',
                          style: TextStyle(
                            fontSize: 22,
                          )),
                    ),
                  ),
                ],
              )
            ],
          ),
        ),
      ),
    );
  }
}
```
