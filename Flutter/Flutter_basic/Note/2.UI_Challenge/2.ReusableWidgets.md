### ResuableWidget 만들기
반복되는 Widget을 분리하여 class를 만들자
- StatelessWidget 상속받아 class 생성. build()함수 override
- 파라미터로 사용될 변수를 인자로 받는 생성자 정의
- VS Code의 자동 추출 기능을 활용해도 됨.
```
반복 위젯 선택 -> cmd + . -> Extract Widget
```
- VS Code에서 위젯 코드 자동 생성
```
flutter stateless widget 타이핑하면 자동 생성해줌
그냥 st라고 타이핑해도 됨
```

```dart
class Button extends StatelessWidget {
  final String text;
  final Color bgColor;
  final Color textColor;

  const Button({
    super.key,
    required this.text,
    required this.bgColor,
    required this.textColor,
  });

  @override
  Widget build(BuildContext context) {
    return Container(
      decoration: BoxDecoration(
        color: bgColor,
        borderRadius: BorderRadius.circular(45),
      ),
      child: Padding(
        padding: const EdgeInsets.symmetric(
          vertical: 20,
          horizontal: 50,
        ),
        child: Text(text,
            style: TextStyle(
              color: textColor,
              fontSize: 20,
            )),
      ),
    );
  }
}

//사용
  Button(
    text: 'Transfer',
    bgColor: Color(0x0ff1b33b),
    textColor: Colors.black,
  ),
```
