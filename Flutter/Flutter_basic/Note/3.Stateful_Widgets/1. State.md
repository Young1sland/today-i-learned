### Stateful Widget
- Stateless Widget은 build 메소드를 통해 UI를 출력만 함
- Stateful Widget은 상태를 가지고 있음. 데이터 변화를 UI에 보여줘야 하는 경우 사용
- VS에서는 st라고 타이핑하면 자동 생성 가능
- 또는 StatelessWidget에서 command + '.'하면 StatefullWidget으로 변경하는 기능 제공함

```dart

void main() {
  runApp(const MyApp());
}

class MyApp extends StatefulWidget {
  const MyApp({super.key});

  @override
  State<MyApp> createState() => _AppState();
}

class _AppState extends State<MyApp> {
  int counter = 0;
  void onClicked() {
    setState(() {
      //데이터가 갱신 되었으므로 UI 업데이트 필요함을 알려줌. build()가 다시 실행 됨
      counter = counter + 1;
    });
    //setState()안에 넣지 않아도 되나 가독성을 위해 setState()안에 넣는 것이 좋다
    /* 하기와 같이 해도 UI 새로고침 됨
    counter = counter + 1;
    setState((){});
     */
  }

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        backgroundColor: const Color(0xFFF4EDDB),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              const Text(
                'Click Count',
                style: TextStyle(
                  fontSize: 30,
                ),
              ),
              Text(
                '$counter',
                style: const TextStyle(
                  fontSize: 30,
                ),
              ),
              IconButton(
                iconSize: 30,
                onPressed: onClicked,
                icon: const Icon(Icons.add_box_rounded),
              )
            ],
          ),
        ),
      ),
    );
  }
}
```
