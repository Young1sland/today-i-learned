# Ubuntu에 Redis Server 설치
```shell
$ sudo apt-get update
$ sudo apt-get upgrade
$ sudo apt-get install redis-server
$ redis-server --version
```

### Redis 설정
/etc/redis/redis.conf 파일을 통해 설정 수정한다.
```shell
# redis.conf
daemonize yes
requirepass {password} # 접속 password 설정
bind 0.0.0.0 # 모든 ip로 부터의 접속 허용
maxmemory {bytes} # maxmemory 설정. 500mb, 1gb..
maxmemory-policy {policy} # 데이터 교체 알고리즘
loglevel {loglevel} #default notice
logfile {경로} # default /var/log/redis/redis-server.log
```

### Redis 서비스 시작
```shell
$ sudo systemctl start redis-server
$ sudo netstat -ntlp | grep 6379
$ sudo systemctl enable redis-server.service # 서버 시작 시 자동 실행
```

### WARNING 해결
```shell
#WARNING overcommit_memory is set to 0! Background save may fail under low memory condition 해결
$ sudo sysctl vm.overcommit_memory=1 # /etc/sysctl.conf에 추가, reboot
```

### Redis 접속 확인
```shell
$ redis-cli -h {ip_address} -p 6379 -a {password}
```
