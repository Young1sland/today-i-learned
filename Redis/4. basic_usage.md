### PipeLine

м„ңлІ„лЎң multiple commandsлЎӨ н•ң requestм—җ ліҙлӮҙкі  responsesлҠ” лӮҳмӨ‘м—җ л°ӣлҠ” кІғ

```py
# pipeline_ex.py

# нҢҢмқҙн”„лқјмқё мӮ¬мҡ©
pipe = r.pipeline()

# Queue commands in the pipeline
pipe.set('name', 'Alice')
pipe.set('age', 30)
pipe.set('country', 'USA')
pipe.mset({'state': 'california', 'city': 'san francisco'})
pipe.mget(['state'])

# Execute the pipeline and fetch results
print(pipe.execute())
"""
[True, True, True, True, ['california']]
"""

# Get values using pipeline
values = r.mget('name', 'age', 'country', 'city')

# Print values
print("Name:", values[0])
print("Age:", values[1])
print("Country:", values[2])
print("City:", values[3])
```

### Set(м§‘н•©)

Uniqueн•ң elements, мҲңм„ңм—ҶлҠ” collection. hash tableмқ„ мӮ¬мҡ©н•ҳлҜҖлЎң л№ лҘҙлӢӨ.

```py
# set_ex.py

# SADD: setм—җ element м¶”к°Җ
r.sadd('myset', 1, 2, 3, 4, 5)

# SISMEMBER : Check if an element exists in the set
print(r.sismember('myset', 3))  # 1

# SREM: Remove elements from a set
r.srem('myset', 2, 4)

# SMEMBERS: Get all members of a set
print(r.smembers('myset'))  # {'1', '3', '5'}

# SCARD: Cardinality. м§‘н•©мқҳ мӣҗмҶҢ мҲҳ кө¬н•Ё
print(r.scard('myset'))  # 3

# Create two sets
r.sadd('set1', 1, 2, 3, 4)
r.sadd('set2', 3, 4, 5, 6)

# SINTER: Set intersection
intersection = r.sinter('set1', 'set2')
print("Intersection:", intersection)  # Intersection: {'3', '4'}

# SUNION: Set union
union = r.sunion('set1', 'set2')
print("Union:", union)  # Union: {'1', '5', '6', '2', '3', '4'}

# SDIFF: Set difference
diff = r.sdiff('set1', 'set2')
print("Difference:", diff)  # Difference: {'1', '2'}

# SUNIONSTORE: unionмқ„ мӢӨн–үн•ңлӢӨмқҢм—җ кІ°кіјлҘј мІ«лІҲм§ё parameterмқҳ setм—җ м ҖмһҘ.
r.sunionstore('union_result', 'set1', 'set2')
print("Union Result:", r.smembers('union_result'))  # Union Result: {'5', '6', '1', '4', '2', '3'}

# SDIFFSTORE: diff мӢӨн–ү нӣ„ м ҖмһҘ
r.sdiffstore('diff_result', 'set1', 'set2')
print("Difference Result:", r.smembers('diff_result'))  # Difference Result: {'2', '1'}

# SINTERSTORE: көҗм§‘н•© мӢӨн–ү нӣ„ кІ°кіј м ҖмһҘ
r.sinterstore('inter_result', 'set1', 'set2')
print("Intersection Result:", r.smembers('inter_result'))  # Intersection Result: {'3', '4'}
```

### Sorted Set

unique elements, мҲңм„ңк°Җ мһҲлҠ” collection. к°Ғ memberлҠ” мҲңм„ңм—җ мҳҒн–Ҙмқ„ мЈјлҠ” scoreлҘј к°Җм§„лӢӨ.

```py
# sorted_set_ex.py

# cleanup the sorted set named 'leaderboard'
r.delete('leaderboard')

# ZADD: л©ӨлІ„лҘј м¶”к°Җн•ңлӢӨ. л©ӨлІ„лҠ” к°’мқ„ к°Җм§Җл©° мқҙм—җ л”°лқј м •л ¬лҗңлӢӨ.
r.zadd('leaderboard', {'Alice': 100, 'Bob': 200, 'Carol': 150, 'Tom': 300})


# ZREM: л©ӨлІ„ мӮӯм ң
r.zrem('leaderboard', 'Carol')


# ZSCORE: л©ӨлІ„мқҳ scoreлҘј к°Җм ёмҳЁлӢӨ.
alice_score = r.zscore('leaderboard', 'Alice')
print("Alice's score:", alice_score)
# 100.0


# ZCARD: л©ӨлІ„ мҲҳ к°Җм ёмҳЁлӢӨ.
leaderboard_size = r.zcard('leaderboard')
print("Number of members in the leaderboard:", leaderboard_size)
# 3


# ZCOUNT: Score лІ”мң„ лӮҙмқҳ л©ӨлІ„ мҲҳ кө¬н•ҳкё°
members_in_range_count = r.zcount('leaderboard', 120, 200)
print("Number of members with scores between 120 and 200:", members_in_range_count)
# Number of members with scores between 120 and 200: 1


# between 120 and 200(exclusive)
members_in_range_count = r.zcount('leaderboard', 120, '(200')
print("Number of members with scores between 120 and 200(exclusive):", members_in_range_count)
# (exclusive): 0


# ZRANGE: index лІ”мң„мқҳ л©ӨлІ„,к°’мқ„ к°Җм ёмҳЁлӢӨ. -1мқҖ л§Ҳм§Җл§ү index лң»н•Ё
leaderboard = r.zrange('leaderboard', 0, -1, withscores=True)
print("Leaderboard after removing Carol:", leaderboard)
# [('Alice', 100.0), ('Bob', 200.0), ('Tom', 300.0)]


# ZRANGEBYSCORE: scoreмқҳ min, max мӮ¬мқҙмқҳ л©ӨлІ„ кө¬н•ҳкё°
members_in_range = r.zrangebyscore('leaderboard', min=80, max=200)
print("Members in range 80-200:", members_in_range)
# ['Alice', 'Bob']


# ZREVRANGEBYSCORE: scoreмқҳ min, max мӮ¬мқҙмқҳ л©ӨлІ„ кө¬н•ҳкё°. лӮҙлҰјм°ЁмҲң
members_in_range_reverse = r.zrevrangebyscore('leaderboard', min=80, max=200)
print("Members in range 80-200 (in reverse order):", members_in_range_reverse)
# ['Bob', 'Alice']


# ZPOPMIN : lowest scoreмқҳ л©ӨлІ„лҘј м ңкұ°н•ҳкі  scoreмҷҖ к°ҷмқҙ лҰ¬н„ҙн•ңлӢӨ.
lowest_member = r.zpopmin('leaderboard')
print("Member with the lowest score:", lowest_member)
# [('Alice', 100.0)]


# ZPOPMAX: highest scoreмқҳ л©ӨлІ„лҘј м ңкұ°н•ҳкі  scoreмҷҖ к°ҷмқҙ лҰ¬н„ҙн•ңлӢӨ.
highest_member = r.zpopmax('leaderboard')
print("Member with the highest score:", highest_member)
# [('Tom', 300.0)]

# ZINCRBY: л©ӨлІ„мқҳ scoreм—җ к°’мқ„ лҚ”н•ңлӢӨ.мЎҙмһ¬н•ҳм§Җ м•Ҡмңјл©ҙ мғқм„ұн•ңлӢӨ.
new_score = r.zincrby('leaderboard', 50, 'Bob')
print("New score of 'Bob':", new_score)
# 250.0

new_score = r.zincrby('leaderboard', 75, 'Dan')
print("New score of 'Dan':", new_score)
# 75.0

```
