### MAC OS에서 oracle 설치하기
```sh
$ docker pull gvenzl/oracle-xe
$ docker images # 존재하는 docker image 확인
$ colima start --arch x86_64 --memory 4 # 2023 9월 현재 appleM chip에서 실행 불가해서 실행하게 하기 위함
$ docker run -d -p 1521:1521 -e ORACLE_PASSWORD=abc1234 -v oracle-volume:/opt/oracle/oradata gvenzl/oracle-xe # oracle 실행
$ docker exec -it dd6d3d62769a sqlplus # oracle 접근, ID: system, 기본 passwd : oracle. 위에서 ORACLE_PASSWORD로 변경했다면 변경한 password로 입력
```
