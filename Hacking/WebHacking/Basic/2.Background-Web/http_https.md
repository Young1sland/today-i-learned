## 유니코드 인코딩
- 문자를 포함한 데이터는 0과 1로 구성 됨. 문자를 표현하는 약속을 인코딩 표준이라고 함.  
- 유니코드는 모든 문자를 하나의 표준에 담겠다는 목표로 제정되었음.  
  한 문자는 최대 32개의 비트로 표현 됨. (약 42억개 표현 가능)  

## Well-known Port(Privileaged Port)
- 0 ~ 1023번 포트를 이름.(22번 ssh, 80번 http, 443번 https)

## CRLF
Line Feed(LF) : 아래 줄로 이동 하는 것  
Carrige Return(CR) : 왼쪽 끝으로 밀어주는 것
