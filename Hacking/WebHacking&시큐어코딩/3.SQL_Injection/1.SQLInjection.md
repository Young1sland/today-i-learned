### 취약점 판별 방법
1. 사용자 입력(파라미터) - 숫자형  
   산술 연사자를 통해 판별 가능  
   http://127.0.0.1:8888/insecure_website/index.php?page=view&idx=2+1  
   3번 idx의 게시글 조회 됨.  
   $query = "select * from board where idx={$idx}";  
   이렇게 넣어봐도 됨 (case when 1=2 then 1 else 2 end)   
2. 사용자 입력(파라미터) - 문자형  
   연결 연산자 사용  
   - 게시글 검색 예시 
     - te' 'st 입력 -> test 검색 됨.  
      `select 'te' 'st';`
     - test%' and '%'=' 입력 -> test검색 됨. %가 붙어 있는 상태이므로  
      `select 'test%' and '%'='%';`  
     - test%' and 1=1# -> test 검색 됨. #으로 이후 주석 처리  
      `select 'test%' and 1=1#%';`
