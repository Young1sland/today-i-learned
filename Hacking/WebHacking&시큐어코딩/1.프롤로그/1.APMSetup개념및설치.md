### APM Setup 
Apache+PHP+MySQL 한번에 설치해 줌
- PHP.ini 경로 
  /Applications/MAMP/bin/php/php7.4.21/conf/php.ini
- Apache root 경로 
  /Applications/MAMP/htdocs

### MYSQL 초기 비번 업데이트
```sh
mysql> update mysql.user set password=password('password') where user='root';
mysql> flush privileges;
```

### Burp Suite 설치
필터링 하고자하는 IP,domain을 제한할 수 있다.
 - Proxy -> Options
 - Intercept Client Requests, Intercept Server Responses
 - ADD -> Match Condition 설정  
   ex) 127.0.0.1|192.168.0.1 ('|'는 or의 의미)
