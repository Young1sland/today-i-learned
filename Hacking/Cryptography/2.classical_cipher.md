# 고전 암호
고성능 연산장치 발명 전 간단한 암복호화. 일반적으로 치환(Substitution),전치(Transposition)방법으로 설계 됨  
치환은 평문의 문자를 다른 문자로 바꾸는 것을 말하며 전치는 평문 문자들의 위치를 바꾸는 것을 말함

## 단일 문자 치환 암호
평문의 각 문자를 약속된 다른 문자로 치환하는 암호. 복호화를 위해 치환의 대응 관계는 일대일 대응.

### 카이사르 암호
- 줄리어스 카이사르가 사용한 암호. 평문의 각 알파벳을 정해진 횟수만큼 다음 순서에 해당하는 알파벳으로 치환
- 'Z'의 다음 알파벳은 'A'로 정의. 복호화 시 암호문의 각 문자를 다시 정해진 횟수만큼 이전 알파벳으로 치환시켜 평문을 구함
- 알파벳을 밀어낸 횟수만 알면 해독할 수 있음. 알파벳을 밀어낸 횟수를 키(key)라고 한다면 알파벳 수만큼 가능한 키의 가짓수는 26개

### 춤추는 인형과 코드북 암호
- 셜록 홈즈에 나온 춤추는 인형 암호. 사람 한명이 글자 하나에 대응되지만 규칙성이 없어 평문을 쉽게 유추할 수 없음.
- 모든 알파벳을 서로 다른 기호와 무작위로 일대일 대응시켜 치환하면 키 공간의 크기는 26!. 현대 컴퓨터로도 전부 탐색하기 어려움
- 언어가 지닌 통계적 특성이 유지된다는 단점. 영어에서 'e'가 많이 사용되므로 이러한 특성으로 추측 가능

### 난수표, 코드북 이용한 단일 치환 암호
- 송신자,수신자가 책을 정하고 송신자가 책의 페이지 x와 단어의 인덱스 y를 보내면 수신자가 해당되는 단어 확인하여 해독

## 다중 문자 치환 암호
평문의 한 문자가 암호문에서 여러 종류의 문자로 치환될 수 있음. 대표적으로 비제네르 암호(Vigenere cipher)가 있음.

## 전치 암호
평문을 구성하는 문자들의 순서를 재배열하여 암호문을 만든다. 평문을 정해진 길이의 블록들로 나누고, 규칙을 적용하여 블록 안의 문자들을 재배치. 대표적으로 스키테일(Scytail)암호가 있음.

## 고전 암호 공격
대표적으로 전수 키 탐색 공격(Exhaustive key search attack)과 빈도수 분석(Frequency analysis)이 있음.

### 전수 키 탐색 공격(Exhaustive key search attack)
- 평문과 암호문을 알 때, 키 공간을 전부 탐색하며 주어진 암호문과 같은 암호문을 생성하는 키를 찾는 방법
- 키 공간의 크기가 작다면 빠른 시간안에 키를 찾고, 암호 해독 가능.
- 단일 치환 암호인 카이사르 암호는 키 공간이 26으로 매우 작기 때문에 전수 키 탐색 공격에 취약.

### 빈도수 공격
- 단일 치환 암호는 평문의 문자와 암호문의 문자가 항상 일대일 대응을 이루기 때문에 평문의 통계적 특성이 유지 됨. 예를 들어, 알파벳 사용 빈도
- 알파벳 사용 빈도 같은 특성으로 추측하여 암호문을 복호하는 것을 뜻함.
- 다중 치환 암호는 이러한 통계적 특성이 사라지기 때문에, 빈도수 공격으로부터 비교적 안전하다고 알려져 있음.
