## SageMath(Sage)
CAS(Computer algebra system)라고도 불리는 컴퓨터용 수학 소프트웨어 중 하나.         
다양한 기능이 구현되어 있고, Python 위에서 동작   
[SageMath Git](https://github.com/3-manifolds/Sage_macOS/releases)

### SageMath 활용
- 셸에서 sage 명령어로 사용할 수 있고, `sage *.sage` 명령어를 통해 작성한 코드를 실행시킬 수도 있음.  
- Conda를 이용하여 설치하였을 경우에는 Python에서 `from sage.all import *` 를 이용해 사용하는 것 또한 가능
- Python에 설치된 패키지와 SageMath에 설치된 패키지는 개별적
- SageMath에 패키지 설치는 `sage --pip install *`  
- Python에서 ^ 연산자는 xor 연산을 의미하지만, SageMath 셸에서는 거듭제곱, 즉 **과 같음. XOR 연산은 ^^을 이용

### pwntools
바이너리 분석, 소켓 통신을 편하게 해주는 여러 기능을 가지고 있음.  
`sage --pip install pwntools`로 설치

### tqdm 패키지
진행도, 예상시간 표시
- arr 순회 시 `for val in tqdm(arr):` 사용하면 진행도 학인 가능
- iterator 오브젝트 순회와 같이 길이가 정의되지 않고 __next__ 통해 다음 원소만 정의된 경우 `tqdm(iterator, total=100)`과 같이 total인자 사용해 최대 길이 지정 가능
```py
from tqdm import tqdm, trange
val = 0
for i in trange(10000000):
    val += 3

100%|██████████████████████████| 10000000/10000000 [00:01<00:00, 9315038.92it/s]
val
30000000
```
