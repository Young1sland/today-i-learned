# Interface

### 상수 필드와 추상 메소드
- 인터페이스에 선언된 필드는 모두 public static final 특성을 갖는다. 키워드를 생략해도 컴파일 과정에서 붙게 된다.
- 구현 클래스가 재정의해야 하는 public abstract 메소드를 멤버로 가질 수 있으며 키워드를 생략해도 컴파일 과정에서 붙게 된다.
```java
public interface RemoteControl {
    //public static final field
    int MAX_VALUE = 10;
    int MIN_VALUE = 0;

    //public abstract method
    void turnOn();   
    void setVolumn(int volume);
}

public class Television implements RemoteControl {
    private int volume;
    @Override
    public void turnOn(){
        System.out.println("TV");
    }

    @Override
    public void setVolumn(int volume){
        System.out.println("volume");
    }
}

RemoteControl rc = new Television();
```
