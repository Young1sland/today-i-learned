# 스트림

## 스트림
- 컬렉션 및 배열의 요소를 반복 처리하기 위해 스트림 사용할 수 있음.
- 내부반복자이므로 Iterator보다 처리 속도가 빠르고 병렬 처리에 효율적    
  *외부반복자 : for문과 iterator과 같이 컬렉션의 요소를 컬렉션 바깥쪽으로 반복해서 가져와 처리    
   내부반복자 : 요소 처리 방법을 컬렉션 내부로 주입시켜서 요소 반복 처리 
- 람다식으로 다양한 요소 처리 정의할 수 있고 중간 처리, 최종 처리를 수행하도록 파이프 라인 형성할 수 있음.
```java
Set<String> set = new HashSet<>();
set.add("Hong");
set.add("Shin");
Stream<String> stream = set.stream(); //Stream 객체 얻음
stream.forEach(name->System.out.println(name));
```
### 내부 반복자
- 개발자 코드에서 제공한 데이터 처리 코드(람다식)을 가지고 컬렉션 내부에서 요소 반복 처리
- 멀티 코어 CPU를 최대한 활용하기 위해 요소들을 분배시켜 병렬 작업 가능
```java
List<String> list = new ArrayList();
list.add("Hong");
list.add("Kim");
Stream<String> parallelStream = list.parallelStream(); //병렬 처리 스트림 얻음
parallelStream.forEach(name-> {
    System.out.println(name+": "+Thread.currentThread().getName())
});
/* 출력
Hong : main
Kim : ForkJoinPool.commonPool-worker-1
*/
```

### 스트림 파이프라인
- 스트림은 하나 이상 연결될 수 있음
- 중간스트림에서 필터링, 매핑,정렬 등을 수행. 최종 처리는 요소 반복, 카운팅, 총합, 평균 등의 작업 수행
- 스트림 파이프라인 구성할 때는 반드시 최종 처리 부분이 있어야 함.
```java
//Student 스트림 얻기 -> score 스트림으로 변환 -> score 평균 구함
Stream<Student> studentStream = list.stream();
IntStream scoreStream = studentStream.mapToInt( student -> student.getScore());
double avg = scoreStream.average().getAsDouble();

//메소드 체이닝 패턴 이용
double avg = list.stream()
            .mapToInt(student->student.getScore())
            .average().getAsDouble();
```

## 리소스로부터 스트림 얻기
- java.util.stream 패키지에 스트림 인터페이스가 있음. BaseStream 인터페이스를 부모로 한 자식 인터페이스들이 있음
- Stream(객체 요소 처리), IntStream, LongStream, DoubleStream..

### 컬렉션으로부터 스트림 얻기
stream(),parallelStream()메소드로 List,Set 인터페이스를 구현한 모든 컬렉션에서 객체 스트림 얻을 수 있음.
```java
public class Product {
  private int pno;
  private String name;
  public Product(int pno, String name){
    this.pno = pno;
    this.name = name;
  }
  @Override
  public String toString(){
    return new StringBuilder().append("pno:"+pno+",")
    .append("name:"+name).toString();
  }
}
public class Example {
  public static void main(String[] args){
    List<Product> list = new ArrayList<>();
    Product product1 = new Product(1,"product1");
    Product product1 = new Product(2,"product2");
    list.add(product1);
    list.add(product2);
    Stream<Product> stream = list.stream();
    stream.forEach(p->System.out.println(p));
  }
}
```

### 배열로부터 스트림 얻기
java.utils.Arrays 클래스 이용
```java
String[] strArr = {"Hong", "Kim"};
Stream<String> strStream = Arrays.stream(strArray);
strStream.forEach(item->System.out.print(item+","));
System.out.println();

int[] intArr = {1,2,3};
IntStream intStream = Arrays.stream(intArr);
intStream.forEach(item->System.out.print(item+","));
System.out.println();
```

### 숫자 범위로부터 스트림 얻기
- IntStream, LongStream의 정적 메소드 range(), rangeClosed() 이용
- 첫번째 매개값은 시작 수, 두번째 매개값은 끝 수. 끝 수 포함하지 않으면 range(), 포함하면 rangeClosed()
```java
static int sum;
IntStream stream = IntStream.rangleClosed(1,100); //1~100까지
stream.forEach(a->sum+=a);
```

### 파일로부터 스트림 얻기
java.nio.file.files의 lines() 메소드 이용하여 텍스트 파일의 행 단위 스트림 얻을 수 있음
```java
/*data.txt
{"pno":1, "name":"product1"}
{"pno":2, "name":"product2"}
*/
public class Example{
  public static void main(String[] args) throws Exception {
    //data.txt 파일의 경로 객체 얻기
    Path path = Paths.get(Example.class.getResource("data.txt").toURI());
    //path로부터 파일 열고 한행씩 읽으면서 문자열 스트림 생성
    Stream<String> stream = Files.lines(path, Charset.defaultCharset());
    stream.close();
  }
}
```
