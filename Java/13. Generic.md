# Generic

## Generic
제네릭이란 결정되지 않은 타입을 파라미터로 처리하고 실제 사용할 때 파라미터를 구체적인 타입으로 대체시키는 기능
```java
public class Box<T>
{
    public T content;
}

public class GenericExample {
    public static void main(String[] args){
    Box<String> box1 = new Box<>(); //new Box<String>()
    box1.content = "Hello";    
    }
}
```

## Generic Type
제네릭 타입은 결정되지 않은 타입을 가지는 클래스와 인터페이스를 뜻함.
```java
//Class
public class Product<K, M>{
    private K kind;
    private M model;
    public K getKind(return this.kind);
    public void setKind(K kind){this.kind =kind};
    public void setModel(M model){this.model = model};
}
public class Car {}
public class GenericExample {
    public static void main(String[] args){
    Product<Car, String> product1 = new Product<>();
    product1.setKind(new Car());
    product1.setModel("SUV");   
    }
}
```

```java
//Interface
public interface Rentable<P>{
    P rent();
}
public class Car {
    public void run(){
        System.out.println("Run");
    }
}
public class CarAgency implements Rentable<Car>{
    @Override
    public Car rent(){
        return new Car();
    }
}
public class GenericExample {
    public static void main(String[] args){
    CarAgency carAgency = new CarAgency();
    Car car = carAgency.rent();
    car.run();
   }
}
```

Type을 명시하지 않으면 Object타입으로 간주한다.
```java
public class Box<T>{
    public T content;
    public boolean compare(Box<T> other){
        boolean result = content.equals(other.content);
        return result;
    }
}
public class GenericExample {
    public static void main(String[] args){
        Box box1 = new Box();
        box1.content = "100";
        Box box2 = new Box();
        box2.content = 100;
        boolean result = box1.compare(box2);//false
   
   }
}
```
## 제네릭 메소드
타입 파라미터를 가지고 있는 메소드. 리턴타입과 매개변수 타입에 사용
```java
//public <A,B,..> 리턴타입 메소드명(매개변수,..){...}
public class Box<T>{
    private T t;
    public T get(){return t;}
    public void set(T t){this.t = t;}
}
public class GenericExample {
    public static <T> Box<T> boxing(T t){
        Box<T> box = new Box<T>();
        box.set(t);
        return box;
    }
    public static void main(String[] args){
        Box<Integer> box1 = boxing(100); //T를 Integer로 대체
        System.out.pringln(box1.get());   
   }
}
```
## 제한된 타입 파라미터(Bounded type parameter)
특정 타입과 자식 또는 구현 관계에 있는 타입만 대체할 수 있는 타입 파라미터
```java
//public <T extends 상위타입> 리턴타입 메소드명(매개변수,...){}
public class GenericExample {
    public static <T extends Nunber> boolean compare(T t1,T t2){
        System.out.println(t1.getClass().getSimpleName(),+","+t2.getClass().getSimpleName());
        //Number Method 사용
        double v1 = t1.doubleValue();
        double v2 = t2.doubleValue();
        return (v1==v2);
    }
    public static void main(String[] args){
        boolean result = compare(10,20); //Integer타입으로 대체
   }
}
```
## 와일드카드 타입 파라미터
- <? extends Person> Person과 Person 자식 클래스만 가능
- <? super Student> Student와 Student의 부모 클래스만 가능
- <?> 어떤 타입이든 가능
```java
public class Person(){}
public class Worker() extends Person{}
public class Student() extends Person {}
public class HighStudent() extends Student{}

public class Applicant<T>{ 
    public T kind;
    public Applicant(T kind){
        this.kind = kind;
    }
}
public class Course {
    public static void registerCourse1(Applicant<?> applicant){
        //모든 사람 등록 가능
    }
    public static void registerCourse2(Applicant<? extends Student> applicant){
        //Student, HighStudent 등록 가능
    }
    public static void registerCourse3(Applicant<? super Worker> applicant){
        //Worker, Person 등록 가능
    }
}

public class GenericExample {   
    public static void main(String[] args){
        Course.registerCourse1(new Applicant<Person>(new Person()));
        Course.registerCourse2(new Applicant<HighStudent>(new HighStudent()));
   }
}

```
